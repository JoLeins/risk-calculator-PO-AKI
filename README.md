# risk-calculator-PO-AKI
This repository contains code and developed classification models for Post-operative Acute Kidney Injury (PO-AKI) 
as described in "Identification of Acute Kidney Injury after Non-Cardiac Surgery by Machine Learning Leveraging 
a Large Electronic Health Record Database".
Models are provided for a set of preoperative and a set of pre-and perioperative predictors and are currently 
intended for research purposes only.
